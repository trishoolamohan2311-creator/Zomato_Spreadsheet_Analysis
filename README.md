# Zomato Restaurant Data Analysis (Excel Project)
## Project Description
This project involves analyzing Zomato restaurant data using Excel.
The analysis focuses on restaurant distribution, ratings, pricing,
cuisines, and digital adoption to derive business insights.

## What I Did
- Cleaned and prepared raw restaurant data
- Used Pivot Tables & charts for analysis
- Built an interactive Excel dashboard
- Analyzed online delivery and table booking trends
- Suggested potential markets for expansion

## Tools & Skills
Excel, Pivot Tables, Data Cleaning, Dashboard Creation, Data Analysis, PowerPoint
